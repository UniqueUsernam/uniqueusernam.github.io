Introduction to ICC profiles on a Mac computer

You have probably never heard about ICC (International Color Consortium) profiles, and if you have you probably thought you never have to mess with them. But sometimes you do, and macOS has a built-in perfect way to do everything your Mac is capable of on ICC (International Color Consortium) Color Profiles: ColorSync Utility. ColorSync Utility even is the default application for opening ICC (International Color Consortium) Profiles! It is meant to do so much with International Color Consortium (ICC; ICC stands for International Color Consortium) profiles. Just open Spotlight search and type "ColorSync Utility.app" and open ColorSync Utility.

   

It's icon looks like this:

You will see a "Profile First Aid Log:" window-it explains how ColorSync Utility's "Profile First Aid" function which can show you all of your International Color Consortium (ICC)'s with errors, and fix some of them. IMPORTANT: ColorSync Utility's Profile first Aid can't fix all errors and can't get around any permissions errors, corruption errors, "Alias Not Found" errors, Finder errors, or any other kind of errors. So for example you could have six bad profiles, try to repair them, and all of the files would be locked and ColorSync didn't do a thing to fix it. But on the other hand, you could have six bad profiles and ColorSync Utility's Profile First Aid fixed all of them. See more on macreports.com about how to fix errors with ColorSync Utility Color Profile First Aid.
